Youtube music finder an easy to use youtube-dl companion

What's this?

youtube-dl is a command line tool by Ricardo Garcia Gonzalez, to download youtube videos. Considering it also allows converting downloaded videos to mp3 format and it supports loading a list of youtube urls from an external file, it's the perfect tool to download a bunch of music from youtube.

Before youtube music finder

Think about this: you heard some tracks on the radio. You would love to download them all from the tube. So, you would open a text editor, and point your browser to youtube.com. Then you would start searching for each song, copying the resulting url to the text editor, finally saving the file and feeding it to youtube-dl. Quite a long and repetitive process.

With youtube music finder

You can type all the song names you want to download into the "Search terms" box. Then, when you will hit lookup, it will start searching all of them in youtube, presenting you the first result for each song in a cool table. And after hitting "prepare raw", you will have them ready to paste in a text file to feed to youtube-dl!
